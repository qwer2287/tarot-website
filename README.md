# 🔮 命运之轮 · Wheel of Fortune — Tarot Divination Website

西方魔幻神秘风格的塔罗牌占卜网站。

## 快速开始

### 方式一：使用 Node.js 服务器（推荐）

```bash
cd tarot-website
npm install express
node server.js
```

然后在浏览器中打开 `http://localhost:3000`

服务器端会跟踪所有用户的访问次数，管理员可以看到真实的使用人数。

### 方式二：直接打开 HTML 文件

直接用浏览器打开 `public/index.html`。

在此模式下，访问次数存储在浏览器本地（localStorage），管理员看到的访问次数仅为当前浏览器的计数。

## 功能

1. **开幕玄语** — 进入网站时显示一句玄之又玄的话语，5秒后自动消散
2. **78张塔罗牌** — 包含22张大阿尔卡纳和56张小阿尔卡纳（圣杯、宝剑、星币、权杖）
3. **洗牌与扇形展示** — 78张牌随机洗牌后以扇形展开，显示每张牌的正位/逆位状态
4. **提问弹窗** — 抽牌前弹窗让用户输入想问的问题
5. **三牌抽取** — 用户默念问题后从扇形中抽出3张牌（代表过去、现在、未来），其余牌消散
6. **翻牌机制** — 从左到右依次翻牌，翻转动画精美
7. **AI解牌** — 采用总分总结构：先总览回答 → 逐牌解读 → 给出建议
8. **管理员后台** — 点击右下角🌑按钮进入管理员登录，输入管理码 `taroyuan112510` 查看使用人数

## 技术栈

- 后端：Node.js + Express（访问计数）
- 前端：HTML5 + CSS3 + 原生 JavaScript
- 特殊效果：Canvas 粒子背景、CSS 3D 翻牌动画、扇形展开动画
